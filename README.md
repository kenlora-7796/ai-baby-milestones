# AI Baby Milestones Website

This is a complete GitHub Pages-ready website for the AI-Powered Baby Milestone Journal business.

## Files

- index.html
- style.css
- README.md

## How to use in VS Code

1. Open your project folder in VS Code.
2. Replace your current `index.html` with the new `index.html`.
3. Replace your current `style.css` with the new `style.css`.
4. Save both files.
5. Open with Live Server.

## How to publish with GitHub Pages

1. Create a GitHub repository.
2. Upload `index.html`, `style.css`, and `README.md`.
3. Go to Settings > Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select main branch and /root folder.
6. Click Save.
